# Visit Taniti Prototype

This repository contains an interactive website prototype for the D479 User Experience Design project. The prototype redesigns the Taniti tourism website to help visitors quickly find information about activities, lodging, transportation, food and dining, visitor FAQs, and trip planning.

## Live Prototype

GitHub Pages URL:  
https://GitDevLane.github.io/d479-taniti-prototype/

## Project Purpose

The goal of this prototype is to provide a clearer, more usable tourism website for potential visitors to Taniti. The design supports first-time visitors who need to plan lodging, activities, transportation, dining, and practical travel details before arriving.

## Pages Included

- Home
- Things to Do
- Lodging
- Transportation
- Food & Dining
- FAQs
- Trip Planning

## Prototype Features

- Full-width responsive layout
- Transparent navigation bar
- Hero sections with large travel imagery
- Clickable navigation and footer links
- Image cards for major planning categories
- Interactive FAQ accordion
- Trip planning checklist
- Responsive layout for smaller screens

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages
